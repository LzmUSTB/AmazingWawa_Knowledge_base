# Search Overlay UI Addendum

The preferred Search Overlay top bar layout is:

```txt
[input bar ----------------------][ QUICK | FULL-TEXT ]
[results ------------------------------------------------]
```

The mode toggle should sit in the same row as the input bar, on the right side.

The existing behavior remains:

```txt
Ctrl + Q opens overlay
Tab toggles Quick / Full-text
Quick search searches structured graph data
Full-text is placeholder until Step 11
```

Do not implement Recent or Pin until Step 12.
