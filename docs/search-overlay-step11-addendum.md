# Search Overlay Step 11 Addendum

## Unified Layout

Preferred top bar layout:

```txt
[input bar ----------------------][ QUICK | FULL-TEXT ]
[results ------------------------------------------------]
```

The mode toggle sits on the same row as the input bar.

## Modes

```txt
Quick:
  structured graph search

Full-text:
  note.md and content-block text search
```

`Tab` toggles modes.

## Keyboard

```txt
Ctrl + Q:
  open / close

Esc:
  close

Tab:
  switch Quick / Full-text

ArrowUp / ArrowDown:
  move selection

Enter:
  open result

Shift + Enter:
  open local graph when supported
```

## Step 11 Non-goals

Do not implement:

```txt
Recent
Pin
search history
semantic search
AI search
scroll-to-highlight inside note
```
