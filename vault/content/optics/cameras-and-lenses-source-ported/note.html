从可调相机开始

先从完整相机的几个可调量建立直觉：aperture 决定进入镜头的光锥大小，focal length 决定视场和物体在 sensor 上的放大程度，focus distance 决定哪一个距离上的物体最清晰。

这些变量不是彼此独立的旋钮；调节其中一个，通常会同时改变亮度、模糊、景深和构图。

后面的章节会把这些效果拆开，从光子记录开始逐步重建整个成像机制。

相机参数联动（camera controls）

先把 camera 看成一个由多个参数共同控制的光学系统。改变 aperture 会影响进入相机的光量和景深；改变 focal length 会改变 field of view，也会改变物体在 sensor 上占据的面积；改变 focus distance 会移动最清晰的距离平面。这个演示用于建立整体直觉，后面会把这些变量逐一拆开。

01

记录光：从 photon 到 pixel

数字相机的 sensor 由许多 photodetector 组成。

每个 detector 只负责把落到自己位置上的 photon 转成电信号；收集到的 photon 越多，像素值越高。

但是，一个裸露的 sensor 并不会自动形成图像，因为每个位置会同时收到来自大量方向的光。

为了让“sensor 上的一个位置”对应“场景中的一小块区域”，必须引入几何约束，这就是 pinhole camera 的作用。

曝光时间与亮度（exposure and brightness）

数字图像从 photon 计数开始。每个 detector 像一个小桶，曝光时间越长，落入桶中的 photon 越多，最终像素越亮。收集太少会 underexpose，画面偏暗；收集太多会 overexpose，高亮区域饱和而丢失细节。

单点 RGB 滤光片（RGB filter）

普通 photodetector 只能测量强度，不能直接知道 photon 的 wavelength。为了得到颜色信息，需要在 detector 上方放置 color filter，让不同位置只接受特定波段。这里的重点是：颜色不是每个 sensor cell 天然知道的，而是通过滤光和后处理推断出来的。

Bayer 阵列（Bayer pattern）

Bayer pattern 使用一个 red、一个 blue 和两个 green filter 组成 2×2 单元。green 采样更多，是因为人眼对绿色附近的亮度变化更敏感，绿色通道也更接近亮度信息。真实相机通常不是每个像素完整记录 RGB，而是先记录稀疏颜色样本。

三维场景（3D scene）

为了讨论成像，需要有一个明确的三维场景：球体、地面和天空背景共同向外反射或发出光。拖动视角可以看到，二维照片只是这个三维光场被相机选取后的一个切片。相机真正记录的不是物体本身，而是从物体表面出发并抵达 sensor 的光线。

裸露传感器的问题（bare sensor）

如果把 sensor 直接暴露在场景中，每个 detector 都会同时接收来自天空、地面和物体多个方向的光。结果不是清晰投影，而是所有方向的强度混合。要成像，必须让 sensor 上的每个位置只接收一小束有几何意义的 ray。

Source Cameras and Lenses Location: 记录光：从 photon 到 pixel / #lens_bare_film

场景中的光线方向（scene rays）

漫反射表面会把入射光散射到许多方向。一个场景点不是只向相机发出一条 ray，而是向外发出大量可能的光路。相机的作用是从这些光路中筛选出能形成稳定对应关系的一部分。

Source Cameras and Lenses Location: 记录光：从 photon 到 pixel / #lens_scene_rays

传感器接收到的光线（sensor rays）

没有 pinhole 或 lens 的限制时，sensor 上的一个点会被许多场景点同时照亮。这样一来，一个 pixel 不再对应一个确定的场景方向，图像结构就会消失。成像系统必须把“位置”和“方向”重新建立对应关系。

图像倒置来自简单的几何交叉：场景上方的 ray 穿过 pinhole 后落到 sensor 下方，左侧的 ray 落到右侧。上下翻转和左右翻转叠加，相当于旋转 180°。实际相机会在显示或保存时把图像再旋转回来。

玻璃：光线为什么会偏折

小孔可以筛选方向，但它牺牲了大部分光线，所以图像容易变暗。

要同时获得足够亮度和可控清晰度，需要让更大的光束进入相机，再把它们重新汇聚到 sensor 上。

透镜能够完成这件事，是因为光在空气和玻璃之间传播时会发生 refraction。

理解玻璃如何改变 ray 的方向，是理解 lens、focus 和 aberration 的基础。

玻璃中的视觉偏移（visual shift in glass）

玻璃会改变光的传播方向，是因为光在玻璃中的传播速度不同于空气。把物体放在玻璃后方时，来自物体的 ray 在界面处偏折，眼睛沿直线反推 ray 时，会看到位置发生偏移。这个现象是 refraction 的可见后果。

进入 / 离开玻璃时的光线偏折（ray bending）

ray 进入玻璃时向法线方向偏折，离开玻璃时再偏折一次。两次偏折的组合决定了穿过玻璃后的出射方向。平行玻璃板会让 ray 产生横向位移；非平行界面则可以改变出射方向。

03

波动视角与 Snell 定律

从 ray 的角度看，refraction 只是光线在界面处改变方向；从 wavefront 的角度看，它来自波在不同介质中的传播速度变化。

当波斜着进入玻璃时，先进入玻璃的一侧会先变慢，波前因此发生旋转，ray 作为波前的法线也随之偏折。

Snell 定律描述的正是这种入射角、折射角与折射率之间的关系。

一维正弦波（1D sine wave）

正弦波用于建立波动直觉：amplitude 表示振动强度，phase 表示波峰波谷的位置随时间推进。虽然几何光学常用 ray 表示光路，但 refraction 的深层原因更容易从 wavefront 的连续性理解。

电磁波示意（electromagnetic wave）

光是一种电磁波。电场和磁场相互垂直，并沿传播方向推进。在相机光学中，我们通常不需要完整求解电磁场，但 wavelength 与介质响应的关系会影响 refraction、dispersion 和 chromatic aberration。

垂直进入玻璃的波（normal incidence）

当波垂直进入玻璃时，传播方向不变，但在玻璃内部速度降低，wavelength 变短。波峰仍然平行于界面，所以没有方向偏折。这个案例说明：速度改变不必然导致方向改变，入射角才是关键。

斜入射波的方向改变（oblique incidence）

斜入射时，波前的一侧先进入玻璃并先减速，另一侧还在空气中继续以较快速度前进。这个速度差会让整个 wavefront 旋转。ray 始终垂直于 wavefront，因此 ray 的方向也发生改变。

二维波前（2D wavefront）

二维 wavefront 让折射几何更直观：界面两侧的波峰必须在边界处保持连续，但介质内 wavelength 更短。为了同时满足连续性和较短 wavelength，传播方向只能改变。

三维波前（3D wavefront）

真实光波在三维空间中传播，wavefront 是面而不是线。ray 可以看作 wavefront 的法线方向。透镜通过连续改变 ray 的方向，使来自同一物点的多条 ray 在成像面附近重新相遇。

Source Cameras and Lenses Location: 波动视角与 Snell 定律 / #lens_wave_3d

波前与光线的关系（wavefront and rays）

wavefront 与 rays 是同一光传播现象的两种表示。wavefront 强调相位面如何变形，ray 强调能量大致沿哪个方向传播。相机成像常用 ray，因为它能直接描述光线如何穿过 aperture、lens 并落到 sensor。

Snell 定律基础（Snell’s law）

Snell 定律把入射角、折射角和 refractive index 联系起来。折射率越高，光在介质中越慢；从低折射率介质进入高折射率介质时，ray 通常向法线靠近。这个规律是设计 lens shape 的数学基础。

操纵光线：从折射面到 lens

透镜设计的核心目标，是把来自同一个场景点、但穿过镜片不同位置的 ray 尽量送到 sensor 上的同一点。

单个折射面只能做有限的方向改变；把曲面做成合适形状，并围绕光轴旋转成 lens 后，才能形成可用的成像系统。

focal length、aperture 和 focus distance 共同决定哪些 ray 被接收、在哪里会聚，以及失焦时形成多大的 blur circle。

用折射面改变平行光线（bend parallel rays）

单个折射面可以把一组 parallel rays 偏折到不同方向。改变 surface parameter 会改变法线方向，从而改变每条 ray 的折射结果。透镜曲面的本质就是让不同高度的 ray 遇到不同法线。

focal length 是 lens 把平行入射光会聚到焦点时，焦点到 lens 的距离。较短 focal length 通常意味着更宽的 field of view；较长 focal length 会缩小视场并放大远处物体。摄影中的“广角”和“长焦”都来自这个变量。

像差：真实 lens 为什么不完美

理想透镜模型假设所有 ray 都能完美会聚，但真实玻璃和真实曲面会引入误差。

spherical aberration 来自球面镜片对不同高度 ray 的会聚差异；chromatic aberration 来自不同 wavelength 的光在同一材料中具有不同 refractive index。

真实相机镜头通常使用多个 lens element 和不同玻璃材料来折中亮度、清晰度、重量、成本与像差控制。

Spherical aberration

spherical lens 容易制造，但球面并不能让所有高度的 ray 精确会聚到同一点。靠近 lens 边缘的 ray 与靠近中心的 ray 焦点不同，就会形成 spherical aberration。aspheric lens 可以减少这种误差，但制造更复杂。

光线高度 / 光圈区域（ray height / aperture region）

棱镜色散（prism dispersion）

白光由不同 wavelength 的光组成，而材料的 refractive index 会随 wavelength 改变。棱镜让不同颜色的光折射角不同，于是白光被分散成彩色光带。这个 dispersion 机制也是彩虹和 chromatic aberration 的根源。

Chromatic aberration

在 lens 中，不同 wavelength 的光可能在不同位置会聚，形成 chromatic aberration。轴向色差会让不同颜色的焦点前后错开，横向色差会让画面边缘出现彩色偏移。真实镜头常用 achromatic doublet 或更复杂的镜片组合来减轻它。

复习问题

为什么小孔相机会形成倒立图像？

来自场景上方的 ray 穿过小孔后落到 sensor 下方，来自左侧的 ray 落到右侧；上下和左右都翻转，所以图像整体倒置。

Source Cameras and Lenses Location: Recording Light / lens_film_invert
为什么增大小孔会让图像更亮但更模糊？

更大的 hole 接收更大的 solid angle，因此进入 sensor 的 photon 更多；但同一个场景点的光也会覆盖更多 pixel，点扩散变大，清晰度下降。

Source Cameras and Lenses Location: Recording Light / lens_hole_solid_angle and lens_hole_sharpness
lens 相比 pinhole 的核心优势是什么？

lens 可以接收比小孔更大的光锥，并通过 refraction 把来自同一场景点的多条 ray 重新会聚到 sensor 附近；因此能同时获得更高亮度和可控清晰度。

Source Cameras and Lenses Location: Manipulating Rays / lens_lens_solid_angle and lens_basic
为什么缩小 aperture 会增加 depth of field？

缩小 aperture 会收窄通过 lens 的光锥。sensor 平面离真正焦点稍微偏离时，窄光锥被切出的 blur circle 更小，所以可接受清晰的距离范围变宽。

Source Cameras and Lenses Location: Manipulating Rays / lens_cone_aperture and lens_cone_dof
Chromatic aberration 的根本原因是什么？

材料的折射率随 wavelength 变化，不同颜色的光经过同一 lens 后会有不同焦点或偏折路径，高对比边缘因此出现彩色偏移。

Source Cameras and Lenses Location: Aberrations / lens_prism and lens_chromatic
