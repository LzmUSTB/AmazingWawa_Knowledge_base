# Improved Context Docs

These Markdown files are intended to be exported every time the app runs **Export Context**.

Recommended placement in the exported context folder:

```text
.kb-ai/context/
  AI_KB_GUIDE.md
  PACKAGE_SCHEMA_GUIDE.md
  NOTE_COMPOSITION_GUIDE.md
  SOURCE_EXTRACTION_GUIDE.md
  NOTE_QUALITY_RUBRIC.md
  RELATION_SEMANTICS_GUIDE.md
  BLOCK_REGISTRY.md
  BLOCK_CREATION_POLICY.md
  CONTENT_BLOCK_USAGE_GUIDE.md
  DECLARATIVE_VISUAL_GRAMMAR_GUIDE.md
  EXPRESSION_VISUALIZER_CONTEXT_GUIDE.md
```

Primary goal: move the AI from merely producing importable packages to producing high-density, reviewable, reusable knowledge notes.
