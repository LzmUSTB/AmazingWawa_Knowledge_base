# Wawa Package Schema Guide

## Archive

Archive extension: `.wawapkg`.

Mimetype file content:

```text
application/x-wawa-kb-ai-import-package
```

Recommended packageId:

```text
wawa-import-YYYYMMDD-topic-name
```

## Root structure

```text
mimetype
manifest.yaml
sources.yaml
patch.yaml
generated/
block-types/
review/
```

## manifest.yaml

```yaml
packageFormat: wawapkg
packageKind: import
schemaVersion: "1.1"
packageId: wawa-import-YYYYMMDD-topic-name
title: Human readable title
description: Short description
status: seed
language: zh-CN
preview:
  mode: in-app
  generatedHtmlPreview: false
```

## patch.yaml

```yaml
operations:
  - type: add_domain
    domain:
      id: computer-graphics
      title: Computer Graphics
      description: Rendering, geometry processing, shaders, and graphics pipelines.
      color: "#00B7FF"
      order: 10

  - type: add_node
    node:
      id: curl-noise
      title: Curl Noise
      domain: computer-graphics
      type: concept
      status: growing
      summary: 从标量噪声场构造旋转向量场，用较低成本生成类似流体的纹理运动。
      aliases: []
      tags: [curl-noise, vector-field, vfx]
    parentId: computer-graphics

  - type: add_edge
    from: curl-noise
    to: divergence-and-curl
    relation: depends-on
```

## Generated node file requirements

For every `add_node`, include:

```text
generated/content/<domain>/<node-id>/meta.yaml
generated/content/<domain>/<node-id>/note.md
```

`meta.yaml` must include:

```yaml
id: curl-noise
title: Curl Noise
domain: computer-graphics
type: concept
status: growing
summary: 从标量噪声场构造旋转向量场，用较低成本生成类似流体的纹理运动。
aliases: []
tags:
  - curl-noise
  - vector-field
  - vfx
sourceIds:
  - source-id
createdAt: "2026-06-11T00:00:00+09:00"
updatedAt: "2026-06-11T00:00:00+09:00"
```

Strict match required:
- `meta.id` must equal `node.id`
- `meta.title` must equal `node.title`
- `meta.domain` must equal `node.domain`
- `meta.type` must equal `node.type`
- `meta.status` must equal `node.status`

Do not omit `title` or `type` from `meta.yaml`.

## Asset placement

```text
generated/content/<domain>/<node-id>/assets/<file>
```

Note reference:

```markdown
![Diagram](assets/diagram.png)
```

Allowed asset types:
- png, jpg, jpeg, webp, gif,
- pdf, txt, md, csv, json, yaml, yml,
- mp3, wav, mp4, webm.

Do not include executable/source files:
- js, ts, vue, css, html, htm,
- exe, dll, bat, cmd, sh, ps1,
- jar, wasm.

## Edge constraints

Allowed `add_edge` relations:
- `depends-on`
- `used-in`
- `compares-with`

Do not use `contains` in `add_edge`. `contains` is generated by `parentId`.

## Review files

Required:

```text
review/import-plan.md
review/validation-checklist.md
```

Recommended:

```text
review/source-extraction.md
review/relation-rationale.md
review/block-decision-report.md
review/unresolved-questions.md
```
