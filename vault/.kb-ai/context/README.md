# Wawa AI Context Export

Start here.

This context export is optimized for AI-generated `.wawapkg` import packages.

Recommended reading order:

1. `AI_CONTEXT.yaml`
2. `RULE_PRIORITY.yaml`
3. `AI_KB_GUIDE.md`
4. `SOURCE_TO_NOTE_GUIDE.md`
5. `PACKAGE_AND_RELATION_GUIDE.md`
6. `NOTE_AND_BLOCK_GUIDE.md`
7. `NOTE_QUALITY_RUBRIC.md`
8. `DOMAIN_INDEX.yaml`, `NODE_INDEX.yaml`, `RELATION_INDEX.yaml`, `CUSTOM_BLOCK_INDEX.yaml`

The Markdown files are intentionally consolidated. Do not infer that missing older guide filenames mean the rules were removed; their content has been merged into these fewer guides.
